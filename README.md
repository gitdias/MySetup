## Get Started üöÄ

**`imback.sh`** is the bootstrap script for MySetup ‚Äî a small helper that brings your preferred configuration back to a fresh system with as little friction as possible.

You can run `imback.sh` directly from the repository using either `curl` or `wget`.


**Using curl:**

```bash
curl -fsSL https://raw.githubusercontent.com/gitdias/MySetup/main/imback.sh | bash
```

**Using wget:**
```bash
wget -qO- https://raw.githubusercontent.com/gitdias/MySetup/main/imback.sh | bash
```
